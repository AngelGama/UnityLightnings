# UnityLightnings

![Alt text](/Screenshots/Screenshot3.jpg "Lightning Procedural generated example")

This 2D Unity project generates storms with up to 5 different types of lightnings using procedural generation.

## Types of lightning
- Bolt
- Branch
- Moving lightning
- Nodes
- Burst
